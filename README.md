## 👋 About

This is my professional portfolio website showcasing my skills, projects, and experience as a **Full-Stack Web Developer**, **Node.js + Next.js Engineer**, and **AI Calling Agent Developer**.

## ✨ Features

- 🎨 **Ultra-Premium Design** - Modern, elegant, and visually stunning
- 🌓 **Dark/Light Mode** - Toggle between themes with localStorage persistence
- 📱 **Fully Responsive** - Perfect on all devices and screen sizes
- ⚡ **Smooth Animations** - Glassmorphism effects and scroll animations
- 🚀 **Fast Performance** - Optimized for speed and SEO
- 💼 **8 Complete Sections**:
  - Hero Section with professional introduction
  - Skills showcase with modern tech stack
  - Featured projects with live links
  - About me with statistics
  - Services offered
  - Client testimonials
  - Contact form and information
  - Professional footer

## 🚀 Quick Start

1. Clone the repository:
```bash
git clone https://github.com/YOUR_USERNAME/portfolio.git
```

2. Open `index.html` in your browser or use a local server:
```bash
npx http-server -p 8000
```

3. Visit `http://localhost:8000`
